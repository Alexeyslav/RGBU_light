Проект предназначен для управления аппаратными RGBU модулями http://github.com/Alexeyslav/RGBU_light .
Коммуникация с железом осуществляется посредством COM-порта на скорости 115200 бод

Проекты написаны на Delphi7 и требуют установки дополнительного компонента Aled для отображения состояния подключения.

Приложения:

RGBU_console - настройка и проверка одиночного RGBU-модуля

    Приложение позволяет протестировать работу модуля и изменить адрес любого из модулей включенных в общую сеть.
    Адрес №255 зарезервирован как широковещательный - по этому адресу на команды(кроме команды изменения адреса) реагируют все модули одновременно.
    Для каждого канала можно

RGBU_controller - Контроллер световых эффектов на базе RGBU-модулей
    Считывает данные с файла pattern.txt каждая строчка которого представляет собой состояние модулей на один временной шаг.
    Имеет формат:
    - все строки начинающиеся с символа ";" являются комментариями
    - пустые строки игнорируются
    - отдельные модули разделяются символом ";"
    - каналы разделяются символом "," в количестве не более 4-х
    - яркость каждого канала - целое число от 0 до 255, допустима запись шестнадцетиричного числа в виде $hh где hh - 16-ричное число.
      любые другие формы записи приравниваются к 0.
    - на данный момент реализована работа только с 8-ю модулями имеющими жестко заданные адреса 1..8.
